# mini-react

催学社 mini-react 练习

## 结课感悟

游戏打卡的方式真的不错，帮助其他人解决问题赚取积分，确实是很激励人的方式。只可惜等我真正开始投入其中并为之努力时，副本也要结束了。——可谓是名副其实的副本，快速结束。

在这个过程中，最让我有收获的，倒不是 mini-react 的编写，而是学习方法，愿与诸君分享。

1. 工作日中，上午学习，下午再写公司的业务代码。
2. 不要边看视频边敲代码，自己独立完成。

第一点实在是有用，早上是一点儿也不想干活的，于是刷刷手机发发呆，一早上的大好时光便浪费了。但用来学习便不错，很适合用来巩固或复习昨天学习的东西，正好不需要看什么视频或书籍，打开 vscode 开干就是了。

第二点便是在要求自己了，“艰难困苦，玉汝于成”。自己独立完成一块逻辑的锻炼，虽称不上逆境上的挫折，但于思维和知识而言，很算得上是一种磨砺。这法子正好可与第一点相结合，学了什么先不急于实践，捋清楚流程，至于实践，等第二天到公司再做。什么，你说摸鱼？学习的事情，怎么能叫摸鱼呢[手动狗头]。
